# ☕ Brew Haven

A modern, responsive coffee shop website showcasing UI/UX design and front-end development using HTML, CSS, and JavaScript.

**🔗 Live Demo:** [zasha-site.github.io/brew-haven](https://zasha-site.github.io/brew-haven/)

## About

Brew Haven is a fictional coffee shop landing page built to demonstrate a warm, modern design system paired with interactive front-end features — from a hidden "Explore Menu" reveal to a receipt-styled shopping cart.

## Features

- 🌙 **Dark Mode** — toggle between light and cozy dark themes
- 🔍 **Live Search** — filter menu items with keyword highlighting
- ☕ **Hidden Menu Reveal** — the full menu (Hot Coffee, Cold Drinks, Desserts, Reviews) stays tucked away until "Explore Menu" is clicked
- 🧾 **Receipt-Style Cart** — a signature paper-ticket cart design with add/remove items
- ♡ **Favorites List** — save items and move them straight to the cart
- 🖼 **Our Space Gallery** — a photo showcase of the café's interior
- 🎉 **Beyond the Cup** — Catering, Events, and Workshops highlight section
- 📍 **Contact Section** — location, hours, and a simple map illustration
- ✨ **Scroll Animations** — sections and images animate smoothly into view as you scroll
- 📱 **Fully Responsive** — works across desktop, tablet, and mobile

## Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom properties (design tokens), Flexbox/Grid, animations
- **Vanilla JavaScript** — no frameworks, no dependencies
