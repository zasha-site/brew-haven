# ☕ Brew Haven

A modern, responsive coffee shop website showcasing UI/UX design and front-end development using HTML, CSS, and JavaScript.

**🔗 Live Demo:** [zasha-site.github.io/brew-haven](https://zasha-site.github.io/brew-haven/)

## About

Brew Haven is a fictional coffee shop landing page built to demonstrate a warm, modern design system paired with interactive front-end features — from a hidden "Explore Menu" reveal to a receipt-styled shopping cart.

## Features
- 🌙 **Dark & Light Theme** — switch between light and cozy dark modes
- 🔍 **Live Search** — filter menu items with keyword highlighting
- ☕ **Hidden Menu Reveal** — discover the full menu by clicking **Explore Menu**
- 🛒 **Receipt-Style Shopping Cart** — add and remove items with a unique receipt-inspired design
- ❤️ **Favorites System** — save favorite items and move them directly to the cart
- 🖼 **Our Space Gallery** — explore the café atmosphere through curated images
- 🍽️ **Catering** — delicious food and handcrafted beverages for every occasion
- 🎉 **Events** — host private celebrations and special gatherings
- ☕ **Workshops** — learn brewing techniques and latte art
- 📍 **Contact Section** — location, opening hours, and map illustration
- ✨ **Smooth Scroll Animations**
- 📱 **Fully Responsive Design**

## Tech Stack

- **HTML5** — semantic page structure
- **CSS3** — custom properties (design tokens), Flexbox/Grid, animations
- **Vanilla JavaScript** — no frameworks, no dependencies
